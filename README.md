YUI — Tied in Bloom 🌸

A boutique gifting studio website for YUI, based in Ranipet · Vellore, offering premium bouquets, hampers, and custom gifting solutions.

🔗 Live site: tied-in-bloom-studio.lovable.app


About
YUI — Tied in Bloom is a florist and gifting brand focused on crafting personalized bouquets, baskets, and hampers for everyday moments and special occasions. This repository contains the source code for the brand's official website.
Tagline: Crafting moments, creating memories.

Features
🌺 Product Showcase — Premium Bouquets, Floral & Fruit Baskets, Luxury Hampers, Custom Creations, Return Gifts, and Corporate Gifting
📸 Gallery — Visual showcase of real product work
💬 Direct Ordering — Click-to-chat WhatsApp integration with QR code for quick inquiries
📱 Responsive Design — Optimized for mobile and desktop
🔍 SEO Ready — Configured meta tags, Open Graph, and Twitter card previews
Tech Stack

Frontend: React + TypeScript
Styling: Tailwind CSS
Build Tool: Vite
Hosting/Deployment: Lovable
